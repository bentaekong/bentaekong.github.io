Multiplayer Scratch Project!

This scratch project was inspired by an online flash retro type game found in Newgrounds.com . The original game is also called Superfighters. This game uses an accumulator to keep track of the total lives of the character. This game also uses mostly recent variables in order to add in gravity. This one above is now the version that allows 2 - 4 players to play at the same time.
